上周朋友圈讨论度最高的话题无疑是红色的iPhone 7。此前一直有消息传苹果会推出一款迎合中国消费者的“中国红”，无论目的是否如此，但中国市场确实对这款颜色更为敏感。因为除了喜庆之外，红色对于中国消费者更有吉祥的用意。再阴谋论一些，苹果发布这款iPhone 7版本的时候，正好CEO库克也在中国访问，更加让人揣度这是一场针对中国市场的“套路”。很明显，苹果在中国市场面临的压力越来越大，来自第三方的Counterpoint数据显示，2016年苹果在中国丢掉了“最畅销单品手机”的宝座。如今时代不同了，苹果一年一迭代的模式越来越难，在iPhone 7发布半年之后，红色版本的再次发布一定会再次引起人们的热议，但是否真能够提振苹果的销量，还要看市场反馈，毕竟很多用户还是非常期待今年下半年的十周年版本的iPhone。

相比红色的iPhone 7，上周财经界最受关注的应该是腾讯的财报——全年收入1519.38亿，年度盈利414.47亿，这是腾讯新交出的成绩单。收入同比增幅25%，拿下708.44亿元的游戏业务自然也成功吸引了绝大多数人的眼球。而在《王者荣耀》、《穿越火线》再次被推上话题中心的时候，腾讯音乐这颗财报中的新彩蛋自然也没有被人们“放过”。现在回过头看，或许正是在“对游戏本身的理解、精细化运营和对用户需求的准确把握”这方面，让腾讯顺利地度过了依赖渠道造成的整个手游市场的中年危机。而两倍增速的腾讯云，刚好也是腾讯与阿里竞争最为激烈的领域，其能否重演微信支付的逆转好戏？这或许将是腾讯与阿里这两大巨头最具有战略意义的比赛。

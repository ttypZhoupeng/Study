    要想这个div的高度100%生效，必须设置其父元素高度为一个有效值，100%也行，其他div下一样

# 3.transition 过渡效果
    要实现鼠标移上移下都有过渡效果，transition属性不要加在:hover属性后面
    加在其需要过渡的元素上面例如 nav:hover 
    不加在该选择器下，而是加在nav下
